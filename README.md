## ⚙️ Features

- 🚫 Auto-delete **abuse words**
- 🔗 Blocks **links & spam**
- 📄 Deletes **document/file messages**
- ✍️ Removes **edited messages**
- 🚨 Notifies owner jab bot group me add ho
- ⚡ Ping command for latency check
